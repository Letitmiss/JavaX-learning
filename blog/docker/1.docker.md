# Docker 简介




# Docker安装

## yum源安装
    yum install -y docker 
    
## 常见命令

开机启动 systemctl enable docker.service
启动    systemctl start docker.service
停止     systemctl stop docker.service
重启    systemctl restart docker.service
查看状态  systemctl status docker.service

查看信息:  docker info
帮助 : docker -help
